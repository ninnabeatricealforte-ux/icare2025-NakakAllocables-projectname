# Dataset 5: PSA Projected Population (2020 CPH)

Source: Philippine Statistics Authority (PSA)  
Dataset Title: Projected Population Based on 2020 CPH by Five-Year Age Group and Sex  
Years Covered: 2020–2030  
Units: Thousands of persons  

## Variables

| Variable | Description |
|---------|-------------|
| sex | Population group: Both Sexes, Male, Female |
| age_group | Age brackets: All Ages, 0–4, 5–9, …, 85+ |
| 2020 | Projected population for year 2020 (in thousands) |
| 2021 | Projected population for year 2021 (in thousands) |
| 2022 | Projected population for year 2022 (in thousands) |
| 2023 | Projected population for year 2023 (in thousands) |
| 2024 | Projected population for year 2024 (in thousands) |
| 2025 | Projected population for year 2025 (in thousands) |
| 2026 | Projected population for year 2026 (in thousands) |
| 2027 | Projected population for year 2027 (in thousands) |
| 2028 | Projected population for year 2028 (in thousands) |
| 2029 | Projected population for year 2029 (in thousands) |
| 2030 | Projected population for year 2030 (in thousands) |
